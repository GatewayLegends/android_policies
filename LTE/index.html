LTE Status Privacy Policy

LTE Status is an app provided by Gateway ICT Solutions ("the Company") to allow a specific group of its employees to track and manage the complaints raised by its customers related to network issues in a specific location.


This Privacy Policy describes how the App collects, uses, and shares information from users (employees).

  1. Information Collected

    The App collects the following types of information:

    • Location data

        The app uses location in foreground to determine the correct location latitude and longitude when the app is in foreground and the Network Status Test has been initiated by the user. Foreground location can be allowed by choosing "Allow While using app" at location access prompt. If location is allowed the app will add the measurements while the app is running. If foreground location is not allowed the drive run cannot be started.

    • User Data

      The app collects user information only when the user starts the Network Status Test – Measurement capabilities by pressing the button [Next] after a specific complaint is selected. This information is used for checking the quality of our network in a specific location (or region) and is stored securely in our databases. The collected information contains information about:

      • Fine location of the user (employee).
      • Mobile serving cell parameters.
      • Mobile neighbor cells parameters.
  2. Information Use

    The information collected by the App is used to provide the sole functionality of sending network parameters and location reports. The information is not used for any other purpose.

  3. Information Sharing

    The information collected by the App is not shared with any third parties.

  4. App Usage

    This application is intended for the use of Newroz, Fastlink, and The Gateway companies' employees only and cannot be used by any other user.

  5. App permissions

    The app uses following permissions:

    • Location – to have access to the device location and to be able to measure mobile serving and neighbor cells while the drive test is started.
    • Read phone state – to read phone state, including the current cellular network information.
