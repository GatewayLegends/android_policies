Drive Test App Privacy Policy

Drive-Test ("the App") is provided by Gateway ICT Solutions ("the Company"). This Privacy Policy describes how the App collects, uses, and shares information from users.

  1. Information Collected

    The App collects the following types of information:

    • Location data

      • Foreground location – the app uses location data to determine location latitude, longitude, and altitude. New measurement report is collected when location is changed with a predefined distance interval. Foreground location access is also needed in order for the app to read mobile network serving and neighbor cells measurements.

      • Background location – the app uses location in background to determine the correct location latitude, longitude, and altitude when the app is in background and the Network Test has been initiated by the user. Background location can be allowed by choosing "Allow all the time" at location access prompt. If background location is allowed the app will add the measurements while the app is in background in the same way as if app is in foreground. If background location is not allowed the drive run cannot be started.

    • User Data

      The app collects user information only when the user starts the Network Test – Measurement capabilities by pressing the button [Start]. This information is used for checking the quality of our network in a specific location (or region) and is stored securely in our databases. The collected information contains information about:

      • Fine location of the user (employee).
      • Mobile serving cell parameters.
      • Mobile neighbor cells parameters.
  2. Information Use

    The information collected by the App is used to provide the core functionality of sending network parameters and location reports. The information is not used for any other purpose.

  3. Information Sharing

    The information collected by the App is not shared with any third parties.

  4. App permissions

    The app uses following permissions:

    • Location – to have access to the device location and to be able to measure mobile serving and neighbor cells while the drive test is started.
    • Read phone state – to read phone state, including the current cellular network information.
    • Send Notifications – to show the progress of the Newtork Test in the Notifications Panel.
