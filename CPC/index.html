CPC Privacy Policy

CPC (Cell Physical Configurations) is an app provided by Gateway ICT Solutions ("the Company") to allow its employees update the changes they make to the physical properies on the network sites and cells so that an admin can approve or reject the request


This Privacy Policy describes how the App collects, uses, and shares information from users (employees).

  1. Information Collected

    The App does not collect any type of information.

  2. Information Use

    No information gets collected.

  3. Information Sharing

    No information gets collected to be shared.

  4. App Usage

    This application is intended for the sole use of Newroz, Fastlink, and The Gateway companies' employees and cannot be used by any other user.

  5. App permissions

    The app doesn't use any permissions.
